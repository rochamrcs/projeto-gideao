# Projeto Gideão

O Gideão é um sistema de gerenciamento de produção e logística que tem como objetivo gerenciar de forma simples três setores: produção, armazenamento e carregamento.

### Motivação e objetivos

A motivação para o desenvolvimento deste sistema é a aplicação do conhecimento adquirido em um problema real do dia-a-dia. O objetivo é solucionar a principal dor da operação logística aonde trabalho.


### Tecnologias utilizadas

* Python
* Django


# Funcionalidades

#### Controle de Estoque

* Armazenamento
* Histórico de movimentações
* Status da ocupação
